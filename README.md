# Sistema de Aprobaciones

Aplicación web para gestionar solicitudes de aprobación en el CoE de desarrollo.

## Características
- Crear solicitudes de aprobación
- Gestión de flujos de aprobación
- Historial de decisiones
- Notificaciones

## Tecnologías
- React + TypeScript
- Tailwind CSS
- Node.js (backend pendiente)
